# Reinforcement Learning Solutions
The idea behind this reposity is to build Reinforcement Learning solutions to different type of games / environments.

### Algos:
- **DDPG**
  * MontainCarContinuous
  * Bipedal Walker
  * Pendulum  
  
  `Execute python ddpg.py -env Pendulum-v0 -t 200`
  
  ![pendulum-v0](https://user-images.githubusercontent.com/20289509/58998673-53cadc00-87d8-11e9-85a3-f0a6f3a566b0.png)
  
- **REINFORCE**
  * Cartpole
  
- **Q-Learning**
  * TicTacToe
  
- **DQN**
  * Cartpole
  * Breakout and Pong
    * Solution done with [Samuel Heinrichs](https://github.com/samuelhei)
